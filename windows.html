The Windows Guide

Install on Windows.

Three clicks, one friendly warning to wave past.
macOS Windows Linux
Download for Windows (.exe) 64-bit installer · Windows 10 & 11
1

Run the installer

Open the Litranite_1.0.9_x64-setup.exe you downloaded (usually in your Downloads folder). Double-click it to start.

2

Wave past the blue SmartScreen box

Because Litranite is a free student project and isn’t code-signed yet, Windows shows a blue “Windows protected your PC” screen. This is routine for new apps — not a virus warning. Here’s exactly what to click:

Windows protected your PC
Microsoft Defender SmartScreen prevented an unrecognised app from starting. Running this app might put your PC at risk.
More infoclick this first
App: Litranite_1.0.9_x64-setup.exe
Publisher: Unknown publisher
Run anywaythen this Don’t run

Click More info → then the Run anyway button appears. That’s the whole trick.

3

Finish the setup wizard

Follow the small, polite prompts — Next → Install → Finish. When it’s done, open Litranite from the Start menu. No account, no tour, no telemetry pop-ups.

You’re ready to read.

Drop in a PDF, start highlighting, and everything stays in one quiet place.
