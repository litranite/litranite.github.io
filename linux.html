The Linux Guide

Install on Linux.

One portable file, or your package manager.
macOS Windows Linux
Download AppImage prefer packages? .deb  ·  .rpm
1

Download the AppImage

Grab the .AppImage above. It’s a single self-contained file — nothing to install system-wide, and it runs on most distributions.

2

Make it executable

In a terminal, mark the file as runnable (or right-click → Properties → Permissions → “Allow executing file as program”):

$ chmod +x Litranite_1.0.9_amd64.AppImage
3

Run it

Double-click the file, or launch it from the terminal:

$ ./Litranite_1.0.9_amd64.AppImage

On some systems you may need FUSE: sudo apt install libfuse2.

§

Prefer a package manager?

Install the downloaded package directly instead:

$ sudo dpkg -i Litranite_1.0.9_amd64.deb
$ sudo rpm -i Litranite-1.0.9-1.x86_64.rpm
